How to use your template
  1. Download the template file and save it somewhere on your computer.
  2. Open it in any browser -- Chrome, Safari, Firefox all work fine.
  3. Download your raw export from CD Baby and drop it into the template using the upload button.
  4. Your data populates instantly. Filter, sort, and explore. Export to PDF whenever you need a clean statement.
  5. Next month, just drop in a new export. The file resets and runs fresh every time.
